# Save the King — Pin Puzzle

A standalone browser game inspired by the pull-the-pin rescue minigames shown in mobile puzzle ads.

## Play

If GitHub Pages is enabled, play it here:

https://cassiebrown324-del.github.io/save-the-king-pin-puzzle/

Or open `index.html` / `save-the-king.html` directly in a browser.

## How to play

Tap/click golden pins in the right order to save the king from lava, snakes, and other hazards.

## Contents

- 5 playable levels
- Water/lava/gold/snake hazards
- No build step or dependencies
- Single-file HTML game
