# Save the King — Royal Rescue Match  Playable match-3 rescue prototype inspired by Royal Match / King’s Nightmare rescue ads.  Play: https://cassiebrown324-del.github.io/save-the-king-pin-puzzle/  July 14 Royal Streak skill-reward upgrade: added a visible 🔥 Royal Streak panel under the Rescue Director. Consecutive successful rescue clears now build a streak, awarding extra coins and Crown Cannon charge, while no-match swaps and hazard surges reset the streak. This keeps the 16-level campaign, Nightmare Briefing, Rescue Director, Crown Cannon, Royal Tools, rockets/rainbows, pin panels, map progression, and synced HTML entrypoints intact.  Controls: tap two adjacent tiles to swap. Match glowing goal tiles to complete each rescue objective. Match-4 creates rockets, match-5 creates rainbow clears, Royal Tools give one-use help, and the Crown Cannon breaks locks/goals/blockers once charged.
